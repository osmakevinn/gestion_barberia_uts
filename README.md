## 🚀 Descripción del Proyecto
Este proyecto es un panel administrativo para una barbería desarrollado en **Java SE con Swing**. Permite la gestión de turnos y servicios, integrando una navegación moderna y persistencia de datos.

## 📂 Estructura del Proyecto
- **Menú Principal:** Logo e ítems de navegación (Barberos, Clientes, Citas).
- **CRUD Usuarios:** Gestión completa de los barberos/empleados.
- **Conexión BD:** Implementación de SQLite/MySQL para el almacenamiento de datos.
- **Salir:** Opción de cierre de sesión seguro.

## 🧰 Lista de Tecnologías Usadas
* **Lenguaje:** Java SE 17+
* **Interfaz:** Java Swing / FlatLaf (para el look moderno)
* **Base de Datos:** SQLite (o la que estés usando)
* **IDE:** VS Code / NetBeans / IntelliJ

## 🔧 Instalación y ejecución
1. Clonar el repositorio.
2. Abrir el proyecto en su IDE de preferencia.
3. Ejecutar la clase `Main.java`.
